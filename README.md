# 客服接待考核系统

用真实售后对话当考题的客服在线考核系统。纯静态单页，部署在 GitHub Pages，打开网址即可参与。

## 在线参与

👉 部署后访问：`https://jintaogao65-stack.github.io/kefu-exam/`

## 考核方式

**写话术 + 对照自评**：
1. 看真实对话和提问，用自己的话写出应对回复；
2. 提交后展示「标准应对 + AI 优化话术 + 评分要点」；
3. 对照评分要点逐项自评勾选，系统自动算分；
4. 全部答完出总分和评级，截图发主管。

共 18 题，每题 10 分，满分 180。成绩仅在本地浏览器显示，不上传服务器。

## 技术

- 纯前端单文件 `index.html`（HTML+CSS+JS，零依赖、零后端），适配手机。
- 题库数据从《客服接待考核库》自动解析（`build.py`），店铺名已脱敏。
- localStorage 存进度，刷新不丢。

## 本地构建

```bash
python -X utf8 build.py   # 解析考核库 → 注入模板 → 生成 index.html
```

> `shoplist.json`（脱敏用的真实店铺清单）和 `tasks/`（内部规划）不纳入版本库。
